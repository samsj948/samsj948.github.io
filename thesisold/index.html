About

Within this project Foraging my goal is to find the interconnectedness of mundane objects and create an archive to house them. Finding these objects and scanning them in, with the limits of an 8 1/2 x 11 scanner, the objects become more uniform. From these scans, creating cyanotypes adds another level of uniformity and presents a more physical process of scanning by using the sun as the source of light. Housing all of these cyanotypes and connections within a website allows the work to be more accessible.
